## About Sanad

Sanad is an emergency response application prototype developed using Axure. The design focuses on high-contrast visuals, minimalist UI, and accessible navigation to ensure usability during critical situations. By prioritizing UX/UI principles and cognitive engineering, the interface aims to provide a seamless and intuitive experience for all users.

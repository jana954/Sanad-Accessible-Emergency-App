# SANAD: Accessible Emergency Assistance Application

An accessible mobile application prototype designed using Axure RP to provide immediate support for individuals with visual, auditory, physical, or cognitive impairments. 

##  Key Features
Zero-Login Architecture: Fast access without traditional sign-up during crises.
Automatic GPS Tracking: Automatically shares location at startup.
Oversized SOS Button: Designed for users with motor difficulties or tremors.
Iconic Communication:Visual cue-driven menu for reporting incidents easily.
